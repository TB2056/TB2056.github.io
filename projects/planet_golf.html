Planet Golf

2D Javascript game made with EaselJS

Planet Golf is a simple 2D golf game made with EaselJS. The game focuses navigating the ball across celestial planets to reach the goal.

This project was made as a final project for a web development course and took roughly 4 days in total to complete. The game features simple animations, particle effects, and gravity-physics systems. The game currently has 7 levels focused around different gravity-based physics interactions.
